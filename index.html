Программа обучения для новичков :
Основы веб-технологий
  • HTML : структура веб-страниц, теги, атрибуты.
  • CSS : стилизация, селекторы, макетирование, адаптивный дизайн.
  • JavaScript : основы языка, DOM, события, асинхронность
Основы React
  • Создание и использование компонентов
  • JSX и его применение
  • Состояние и жизненный цикл компонентов
  • Обработка событий и форм
Углубленное изучение React
  • React Router: навигация и роутинг в приложении
  • Оптимизация производительности: Использование React.memo, useMemo, useCallback и виртуализация списка.
Введение в TypeScript
  • Основы TypeScript: типы данных, интерфейсы, классы
  • Применение TypeScript в веб-разработке, включая React
Redux и Redux Toolkit
  • Основы Redux: хранилище, действия, редукторы
  • Подключение Redux к React-приложению
  • Redux Toolkit: упрощение работы с Redux
  • Асинхронные операции с использованием Redux Thunk или Redux Saga
Практические навыки и
инструменты разработки
  • Инструменты сборки: Webpack, Babel
  • Работа с API и обработка ошибок
  • Тестирование: юнит-тесты, интеграционные тесты (Jest, React Testing Library)
  • Версионный контроль с Git и GitHub
  • Основы деплоя и хостинга веб-приложений

Возможность научиться работать с реальными задачами и проблемами, а также узнать, как решать их в команде.
Плотная работа с GitHub и Code Review
Научитесь работать с системой контроля версий Git, практиковать Code Review и другие процессы, используемые при разработке в команде.
Использование современных технологий и инструментов
Курс будет включать в себя работу с React, TypeScript, Redux Toolkit, Nest, TypeORM, Postgres и Docker.
Под руководством опытного разработчика
Курс проводится опытным разработчиком, который будет наставлять вас на протяжении всего курса и помогать решать возникающие вопросы и трудности.
Работа в команде
Научитесь эффективно работать в команде, решать конфликты и общаться с другими членами команды.
Подготовка к работе в реальной среде
Курс обеспечит вас необходимыми навыками для успешной работы в реальной среде разработки, включая коммуникацию с заказчиком, планирование задач, управление сроками и многие другие.
